使用方法：
Windows 7系统，需要安装 .NET 4.0，Windows 8/8.1/10以上系统则无需安装 .NET 4.0。
1、以管理员身份运行“UnInstall_Service.cmd”清除服务。
2、以管理员身份运行KMSpico_setup.exe安装。
3、请耐心等待……，安装完即可自动激活。

静默参数：
KMSpico.exe /silent

使用建议：
★ 如果杀毒软件报毒，请把安装目录以下文件加入白名单或排除项；
%ProgramFiles%\KMSpico\KMSELDI.exe, Service_KMS.exe, Autopico.exe 
W10TP/8.1/2012R/ServerTP 系统的文件为   %WinDir%\SECOH-QAD.exe.

★ 出错情况下，建议检查日志 文件
Program Files -> KMSpico -> logs
通常都是使用过批量许可密钥导致。


1、关于永久激活
问：这是永久激活的吗？
答：当然，一直都是Windows 7/8/8.1/10所有版本。

2、Pro WMC/核心版
问：它只能激活30天或45天？
答：KMS不可能真正的激活Pro WMC/Core，但可以模拟激活为Win8.1可能30天或45天。

3、关于激活联网
问：需要连接网络吗？
答：不需要。
